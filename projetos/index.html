CSS Browser Selector (http://rafael.adm.br/css_browser_selector)

O CSS Browser Selector é um pequeno javascript que facilita a especificação de CSS nos diferentes navegadores. Fica fácil contornar bugs, definindo regras de CSS que funcionarão somente em determinados navegadores. Além disso, o script também reconhece se o navegador do usuário está com o javascript habilitado ou não e com isso é possível apresentar telas informativas sobre a necessidade de habilitar o javascript para a navegação no site ou na aplicação web.
